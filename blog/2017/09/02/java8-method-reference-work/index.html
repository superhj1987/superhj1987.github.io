[иҜ‘]жӣҙеҘҪең°дҪҝз”ЁJava8дёӯзҡ„ж–№жі•еј•з”Ё

Posted by HJ in dzone, java, java8, translate

еңЁJava8дёӯпјҢдҪҝз”Ёж–№жі•еј•з”Ёйқһеёёз®ҖеҚ•пјҢеҰӮString::isEmptyпјҢдҪҶж— жі•дҪҝз”Ёе®ғеҗҰе®ҡзҡ„ж–№жі•еј•з”ЁгҖӮжң¬ж–ҮеҶ…е®№еҚіеҰӮдҪ•и§ЈеҶіжӯӨй—®йўҳдҪҝеҫ—жҲ‘д»¬иғҪеӨҹжӣҙеҠ е…Ёйқўең°дҪҝз”Ёж–№жі•еј•з”ЁгҖӮ

йҰ–е…ҲзңӢдёҖдёӘдҪҝз”Ёж–№жі•еј•з”Ёзҡ„дҫӢеӯҗпјҡ

дёҠйқўд»Јз Ғзҡ„иҫ“еҮәдёә1пјҢеҚіз©әеӯ—з¬ҰдёІзҡ„ж•°зӣ®гҖӮеҰӮжһңжҲ‘д»¬жғіиҰҒиҺ·еҸ–йқһз©әеӯ—з¬ҰдёІзҡ„ж•°зӣ®пјҢе°ұдёҚиғҪзӣҙжҺҘдҪҝз”Ёж–№жі•еј•з”ЁдәҶгҖӮ

Java8дёӯзҡ„PredicateпјҢжңүpredicate.negate()еҸҜд»ҘиҪ¬жҚўдёәж–ӯиЁҖзҡ„еҗҰе®ҡеҪўејҸпјҢдҪҶString::isEmptyеҚҙж— жі•иҝҷд№ҲеҒҡ(String::isEmpty.negate()жҲ–иҖ…!String::isEmpty)гҖӮеӣ дёәж–№жі•еј•з”Ёе№¶дёҚжҳҜдёҖдёӘlambdaжҲ–иҖ…еҮҪж•°жҺҘеҸЈпјҢе®ғиғҪеӨҹиў«и§ЈжһҗдёәдёҖдёӘжҲ–иҖ…еӨҡдёӘеҮҪж•°жҺҘеҸЈгҖӮеҰӮпјҢString::isEmptyиҮіе°‘еҸҜд»Ҙиў«и§ЈжһҗеҰӮдёӢпјҡ

  • Predicate<String>
  • Function<String, Boolean>

дёәдәҶи§ЈеҶідёҠиҝ°зҡ„й—®йўҳпјҢжҲ‘д»¬еҸҜд»ҘйҖҡиҝҮжҹҗз§ҚжңәеҲ¶жҳҫејҸең°е°Ҷж–№жі•еј•з”ЁиҪ¬жҚўдёәдёҖдёӘеҮҪж•°жҺҘеҸЈпјҡ

йҖҡиҝҮдҪҝз”ЁдёҖдёӘйқҷжҖҒж–№жі•пјҢжҺҘеҸ—ж–№жі•еј•з”ЁеҸӮж•°пјҢиҝ”еӣһдёҖдёӘеҮҪж•°жҺҘеҸЈпјҢеҚіеҸҜе®һзҺ°ж–№жі•еј•з”ЁеҲ°еҮҪж•°жҺҘеҸЈзҡ„иҪ¬жҚўгҖӮжҺҘзқҖпјҢжҲ‘д»¬е°ұеҸҜд»ҘдҪҝз”Ёж–№жі•еј•з”ЁжқҘе®һзҺ°дёҠйқўдҫӢеӯҗдёӯзҡ„иҺ·еҸ–йқһз©әеӯ—з¬ҰдёІзҡ„ж•°зӣ®гҖӮ

иҝӣдёҖжӯҘиҝҳиғҪдҪҝз”Ёеҗ„з§Қз»„еҗҲзҡ„PredicateгҖӮ

з”ұдәҺдёҖдёӘж–№жі•еј•з”ЁеҸҜиғҪдјҡиў«и§ЈжһҗдёәеӨҡз§ҚеҮҪж•°жҺҘеҸЈпјҢеӣ жӯӨеҰӮжһңжҲ‘д»¬е®һзҺ°еҫҲеӨҡеҸӮж•°дёҚеҗҢзҡ„asж–№жі•пјҢйӮЈд№ҲеҫҲе®№жҳ“йҖ жҲҗж··ж·ҶгҖӮжӣҙеҘҪзҡ„ж–№ејҸеҲҷжҳҜеңЁж–№жі•еҗҚдёӯеҠ е…ҘеҮҪж•°еҸӮж•°зҡ„зұ»еһӢжқҘеҢәеҲҶгҖӮ
